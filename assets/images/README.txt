PLACE YOUR LOGO HERE
====================

Please save your MyJanji logo image as:
  myjanji_logo.png

Supported formats: PNG, JPG, SVG (PNG recommended for best quality)

The logo should be:
- High resolution (at least 400x400 pixels recommended)
- Transparent background (PNG with transparency)
- The shield with handshake icon and "MyJanji" text

Once you place the file here, run:
  flutter pub get
  flutter run -d chrome
